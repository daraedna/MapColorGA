INSTRUÇÕES PARA GERAR O EXECUTÀVEL

=== Unix ===
    Se estiver utilizando sistema operacional baseado em Unix,
abra um terminal e rode o comando make, a partir daí é só rodar o executável

=== WINDOWS ===
    Edite o arquivo Makefile para apontar o caminho do seu compilador no Windows.
    Ou se preferir, execute o Qt Creator, abra a projeto e gere o executável por lá
